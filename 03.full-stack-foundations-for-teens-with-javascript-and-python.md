# Full Stack Foundations for Teens with JavaScript & Python (Age 14–18 Years)

## Course Duration

- 8 Weeks

---

# Course Description

Learn core programming concepts using both JavaScript and Python. Students will build interactive web pages (JavaScript) and simple programs or games (Python), gaining strong foundational coding skills for future full-stack development.

---

# Learning Objectives

By the end of this course, students will be able to:

- Understand core programming concepts (variables, conditions, loops, functions)
- Apply concepts in both JavaScript and Python
- Build interactive web elements using JavaScript
- Create simple programs and games using Python
- Develop strong problem-solving skills

---

# Curriculum Outline

## Week 1: Introduction to Programming

### Topics Covered

- What is programming?
- JavaScript: console output
- Python: print statements

### Activity

- “Hello World” in both JavaScript and Python

---

## Week 2: Variables & Data Types

### Topics Covered

- Variables, numbers, strings
- Syntax differences (JavaScript vs Python)

### Activity

- Store and display user information

---

## Week 3: Conditions (Decision Making)

### Topics Covered

- `if/else` statements
- Comparison operators

### Projects

- Python: Simple quiz
- JavaScript: Interactive decision app

---

## Week 4: Loops (Repetition)

### Topics Covered

- `for` loops
- `while` loops

### Projects

- Python: Number game
- JavaScript: Repeating UI updates

---

## Week 5: Functions (Reusable Code)

### Topics Covered

- Function creation
- Function usage

### Projects

- Python: Calculator
- JavaScript: Button-based actions

---

## Week 6: Data & Interactivity

### Topics Covered

- Python: Lists (data storage)
- JavaScript: DOM manipulation

### Projects

- Python: Score tracker
- JavaScript: Interactive webpage

---

## Week 7: Mini Project

### Choose One

- Python game (quiz or guessing game)
- JavaScript interactive webpage
- Python + JS combined project
- Interactive website
- Simple app-style project

---

## Week 8: Final Project

Create a complete project using one or both languages:

- Full interactive web application (JavaScript)
- Python-based program or game
- Combined creative project

---

# Assessment

- Class participation
- Weekly exercises
- Mini project (Week 7)
- Final Project (40%)

---

# Outcome

Students will be able to:

- Understand core programming concepts
- Write code in JavaScript and Python
- Build interactive web pages
- Create simple programs and games
- Develop a small project portfolio
