# Saylani SMIT Summer Program – Little Geniuses

> **“From Computer Basics to AI Creator”**

## Coding, Design & AI Fun Lab for Techno Kids (Age 6–11 Years)

## Learning Goals

By the end of the program, students will be able to:

- Understand computers, software, and AI basics
- Create digital designs using Canva
- Build animations and games in Scratch
- Train simple AI models using Teachable Machine
- Communicate effectively with ChatGPT
- Develop simple games using MakeCode
- Present a final creative tech project

## Final Outcome

Each student will build their own:

- Scratch Game or Story
- **AI** Image Recognition Model
- MakeCode **Games**
- Canva **Design** Portfolio
- ChatGPT Prompt Activities
- Final **Portfolio** Presentation
