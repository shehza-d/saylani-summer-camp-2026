# AI & Game Creators

(For Age 11–13)

## Course Duration

- 8 Weeks (Summer Program)
- 2–3 Sessions per Week
- 60–90 Minutes per Session

---

# Course Description

This course introduces students to game development using Roblox Studio and the Lua programming language. Students will learn how to design interactive 3D environments, write simple scripts, and build their own playable games.

The course focuses on creativity, problem-solving, and hands-on learning through guided projects.

---

# Learning Objectives

By the end of this course, students will be able to:

- Navigate and use Roblox Studio tools
- Understand basic Lua scripting concepts
- Create interactive game elements
- Apply logic and conditions in gameplay
- Design and build simple playable games
- Test, debug, and publish their own games

---

# Course Modules

## Module 1: Introduction to Roblox Studio (Week 1)

### Topics Covered

- Interface (Explorer, Properties, Workspace)
- Basic building tools (move, scale, rotate)

### Activities

- Build a simple obstacle course (obby)

### Outcome

Students can create and modify a basic game environment.

---

## Module 2: Lua Programming Basics (Week 2)

### Topics Covered

- What is scripting?
- Variables and data types (numbers, strings)
- Print statements

### Activities

- Display messages inside the game
- Practice simple Lua scripting

### Outcome

Students understand basic coding structure.

---

## Module 3: Interactivity & Events (Week 3)

### Topics Covered

- Events (Touched, Click)
- Functions

### Activities

- Create interactive parts that respond to players

### Outcome

Students build responsive game elements.

---

## Module 4: Game Mechanics (Week 4)

### Topics Covered

- Score systems
- Leaderboards

### Activities

- Create a coin collection system

### Outcome

Students implement rewards and scoring systems.

---

## Module 5: Logic & Gameplay Systems (Week 5)

### Topics Covered

- Conditional statements (`if/then`)
- Game rules and logic

### Activities

- Build doors, traps, and unlock systems

### Outcome

Students apply logic to control gameplay.

---

## Module 6: User Interface Design (Week 6)

### Topics Covered

- GUI elements (text labels, buttons)

### Activities

- Create a simple start/restart interface

### Outcome

Students design basic user interfaces.

---

## Module 7: Project Development (Week 7)

### Topics Covered

- Combining building and scripting
- Iteration and improvement

### Activities

Develop a final game project:

- Obby
- Coin Collector
- Survival Game

### Outcome

Students build a complete playable game.

---

## Module 8: Testing & Publishing (Week 8)

### Topics Covered

- Debugging basics
- Publishing games on Roblox

### Activities

- Final project presentation

### Outcome

Students publish and showcase their game.

---

# Teaching Methodology

- 70% hands-on learning
- Project-based approach
- Gamified challenges
- Collaborative work

---

---

# Final Outcome

Students will be able to:

- Understand basic Lua programming
- Create interactive game elements
- Build and publish a Roblox game
